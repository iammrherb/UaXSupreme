UaXSupreme Help

UaXSupreme is a comprehensive platform for configuring 802.1X, MAB, RADIUS, TACACS+, and advanced authentication features for all major network vendors.

Select a topic from the left navigation menu to view detailed help information.

Getting Started

  1. Begin by selecting your network vendor and platform in the Vendor Selection section.
  2. Choose authentication methods that you want to implement.
  3. Configure RADIUS and/or TACACS+ servers as needed.
  4. Set up advanced features for your implementation.
  5. Configure interfaces for authentication.
  6. Generate and review the configuration.
  7. Generate documentation for your implementation.

For additional assistance, use the AI Assistant by clicking the robot icon in the top navigation bar.
